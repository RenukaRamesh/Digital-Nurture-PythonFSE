# Python Searching Methods

This repository contains simple Python programs demonstrating two basic searching algorithms.


## 📚 Topic
Python Searching Methods

### 1. Linear Search
- Searches each element one by one.
- Returns the index if the element is found.
- Returns `-1` if the element is not found.

### 2. Binary Search
- Works only on a sorted list.
- Repeatedly checks the middle element.
- Returns the index if the element is found.
- Returns `-1` if the element is not found.
